### java8

openjdk-8-jdk

### 安装 java 方法

```bash
sudo apt-get install openjdk-8-jdk
```

### 查看 java 路径的方法

```bash
which java
```

```bash
ls -lrt /usr/bin/java
```

```bash
ls -lrt /etc/alternatives/java
```

```bash
cd /usr/lib/jvm
```
